# 🚀 StockSense AI – Opportunity Radar for Indian Investors

## 🧠 Problem

Retail investors often rely on guesswork, tips, and scattered information. They lack tools that convert complex stock data into clear, actionable decisions.

## 💡 Solution

StockSense AI is an intelligent system that scans multiple stocks, detects trading signals, and provides **AI-powered insights with actionable recommendations (BUY / WATCH / AVOID).**

---

## ⚙️ Features

* 📊 Multi-stock scanner (NSE stocks)
* 📈 Technical indicators (MA, RSI, Volume)
* 🧠 AI-generated explanations
* 🎯 Action recommendations
* 🔥 Top opportunities ranking
* ⚡ Fast performance with caching

---

## 🏗️ Architecture

* Data Agent → Fetch stock data
* Analysis Agent → Compute indicators
* Signal Agent → Detect opportunities
* AI Agent → Generate explanations
* UI Layer → Streamlit dashboard

---

## 🛠️ Tech Stack

* Python
* Streamlit
* yfinance
* Pandas / NumPy
* OpenAI API (optional)

---

## 📊 Impact

* ⏱️ Reduces analysis time from hours to seconds
* 📈 Improves decision-making using data-driven signals
* 🎯 Makes investing accessible for beginners

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
